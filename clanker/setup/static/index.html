HA Add-on Install Steps

1. In HA: Settings → Add-ons → Add-on Store
2. Click … (top-right) → Repositories
3. Add: https://github.com/nolankramer/clanker
4. Find Clanker in the store and click Install
5. Configure your API keys in the add-on settings
6. Click Start

The add-on auto-installs the custom component, configures HA, and gets the auth token automatically. No manual setup needed.
